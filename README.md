## サイト概要
### サイトテーマ
生活の中、外にいる野良猫と迷い猫の情報共有

### テーマを選んだ理由
猫が好きなため、外の猫をみんなで共有できたり
共有した中で迷い猫で情報を求めている人に繋がって行ければいいな
という思い

### ターゲットユーザ
猫が好きな人、猫を探している人

### 主な利用シーン
かわいい野良猫やユニークな野良猫の共有
迷い猫の情報提供

## 開発環境

- 言語：HTML,CSS,JavaScript,Ruby,SQL
- フレームワーク：Ruby on Rails
- JSライブラリ：jQuery
